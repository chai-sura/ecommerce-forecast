# 📦 E-Commerce Demand Forecasting & Supply Chain Optimization

End-to-end data science project covering demand forecasting, regression analysis, and supply chain optimization using real retail datasets.

## 📊 Project Overview

| Notebook | Focus | Key Output |
|----------|-------|-----------|
| `01_eda.ipynb` | Exploratory Data Analysis | Sales trends, seasonality, price elasticity |
| `02_forecasting.ipynb` | Time Series Forecasting | SARIMA, ETS, Prophet, XGBoost comparison |
| `03_regression.ipynb` | Demand Driver Analysis | ANOVA, OLS, Ridge, Lasso, residual diagnostics |
| `04_supply_chain.ipynb` | Inventory Optimization | EOQ, Safety Stock, ROP, stockout risk |

## 🛠️ Tech Stack

| Layer | Tools |
|-------|-------|
| Language | Python 3.9+ |
| Forecasting | SARIMA, ETS (Holt-Winters), Prophet, XGBoost |
| Regression | Statsmodels (OLS), Scikit-learn (Ridge, Lasso) |
| Supply Chain | EOQ, Safety Stock, Reorder Point |
| Visualization | Plotly, Matplotlib, Seaborn |
| Experiment Tracking | MLflow |
| Statistics | ANOVA, ADF Test, Breusch-Pagan, QQ Plot |

## 🚀 Getting Started

### 1. Clone & install

```bash
git clone https://github.com/chai-sura/ecommerce-forecast
cd ecommerce-forecast
python3 -m venv forecast_env
source forecast_env/bin/activate
pip install -r requirements.txt
```

### 2. Mac only — install libomp for XGBoost

```bash
brew install libomp
```

### 3. Download datasets

| Dataset | Source | Save as |
|---------|--------|---------|
| Store Item Demand | [Kaggle](https://www.kaggle.com/competitions/demand-forecasting-kernels-only/data) | `data/raw/store_demand.csv` |
| Online Retail II | [Kaggle](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci) | `data/raw/online_retail.csv` |

### 4. Run notebooks in order

```bash
jupyter notebook
```

Open and run in sequence:
1. `notebooks/01_eda.ipynb`
2. `notebooks/02_forecasting.ipynb`
3. `notebooks/03_regression.ipynb`
4. `notebooks/04_supply_chain.ipynb`

### 5. View MLflow experiment dashboard

```bash
mlflow ui
```
Open http://localhost:5000
